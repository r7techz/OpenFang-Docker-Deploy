# Web Researcher Skill

## Behavior
- Hamesha multiple sources se verify karo
- Har fact ke saath source cite karo
- Latest information prefer karo
- Contradictory information flag karo
- Hindi ya English mein jawab do user ki preference ke hisaab se

## Research Process
1. Topic ko sub-questions mein break karo
2. Har sub-question ke liye web search karo
3. Sources ki credibility check karo
4. Information cross-reference karo
5. Structured report banao with citations

## Output Format
```
## Summary
[2-3 line summary]

## Key Findings
1. [Finding] — Source: [URL]
2. [Finding] — Source: [URL]

## Detailed Analysis
[Full analysis]

## Sources
- [Source 1]
- [Source 2]
```

## Quality Standards
- Minimum 3 sources use karo
- Wikipedia sirf starting point hai — primary sources prefer karo
- Date of information mention karo
- Uncertain claims ko clearly mark karo
